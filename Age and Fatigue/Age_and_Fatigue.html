Results

Description:

This data set, "Age and Fatigue", includes variables that potentially mediate or moderate the effect of age on listening-related fatigue.

Variables:

  1. ID: Subject identifier.
  2. AudAttention: Auditory attention ability score measured with a dichotic listening task.
  3. Age: Age of subject.
  4. HearImpair: Perceived hearing impairment score.
  5. Memory: Perceived memory ability score.
  6. MoodDisturb: Perceived mood disturbances score.
  7. NeedCognition: Need for Cognition score.
  8. SensorySens: Sensory processing sensitivity score.

Each case represents one subject. All variables are z-standardized. This example demonstrates the use of classical conditional process models with causal implication tests.

Reference:

McGarrigle, R., Knight, S., Hornsby, B. W. Y., & Mattys, S. (2021). Predictors of listening-related fatigue across the adult life span. Psychological Science, 32(12), 1937–1951. https://doi.org/10.1177/09567976211016410

This data set is reused with approval of the authors.

Local tests

We can assess the consistency of the hypothesized causal model with the data by testing its local implications. The causal model implies local relationships, i.e., (conditional) independencies, between the included variables. These (conditional) independencies can be tested in the data.

Each row in the table is an independency implied by the causal model. Some independencies are conditional on Age, others are unconditional. Significant independence tests indicate that the causal structure of the model is not consistent with the data and should be adjusted. For example, the model implies that Age and MoodDisturb are independent, but the data suggests that they are correlated.

Conclusion:

There is no evidence for a moderation of perceived mood disturbance on the effect of age on listening-related fatigue. The analysis suggested a positive indirect effect via perceived hearing impairment. However, conditional independence tests indicated inconsistencies between the tested causal model and the data, so the causal model should be adjusted.
