ADD symptoms

Description:

The full data sample consists of 1162 children who were rated by their parents on 18 5-point Likert-scale (0-4) items about each DSM-5 ADHD symptom. The present file sontains the first 9 items representing the attention deficit symptoms. The prevalence of a symptom is indicated from 0 (never) to 4 (very often).

Variables: ASSK_1 - ASSK_9

For example: "My child has diffculty keeping attention on tasks or play"

The example demonstrates how to obtain estimates for measurement precision that are score-related. Specifically, obtaining the standard error of measurement for each sum score of the data set using several different methods. Based on the literature one would expect the measurement precision to be highest at the low and high end points of the scale and decrease towards the middle.

Reference:

Keulers, E. H. H., & Hurks, P. P. M. (2021). Psychometric properties of a new ADHD screening questionnaire: Parent report on the (potential) underlying explanation of inattention in their school-aged children. Child Neuropsychology, 27(8), 1117â€“1132. https://doi.org/10.1080/09297049.2021.1937975

The table contains the confidence intervals (CI) around each sum score differing for each method. Specifically, the 95% CI for a given sum score contains the range of values that we assume cover the true score in 95% of times when repeatedly constructed in the same way. A CI is constructed using the sem and the z-score from a standard normal distribution, making the CIs normal-theory based Wald-type intervals.

Histogram of Counts per Sum Score Group

Point Plots

The point plots show the sem values for the sum scores for each method. For the IRT method the plot is a line-plot since IRT provides a continuous sum score. The combined plot combines all methods in one plot. Contrary to the other methods the IRT based sem values decrease with higher sum scores (which is in line with theory). IRT is also the method with the strongest assumptions.
