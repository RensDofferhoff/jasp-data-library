Album Sales: Linear Regression

Description:

This fictional data set, "Album Sales", provides factors that may influence album sales.

Variables:

  1. adverts - Amount (in thousands of pounds) spent promoting the album before release.
  2. sales - Sales (in thousands of copies) of each album in the week after release.
  3. airplay - How many times songs from the album were played on a prominent national radio station in the week before release.
  4. attract - How attractive people found the band's image (1 to 10).

This example JASP file demonstrates the use of linear regression. First, we will fit a simple model to see whether the adverts predict album sales. Then, we will test whether adding airplay and attractiveness improves the predictions of the model.

Correlation

Now, let's move towards multiple linear regression. Suppose we want to try to relate the sales not only to the advertisement, but also to the radio play and attractiveness of the band. Again, it is usually a good idea to get some descriptives of the data. In this case, we might be especially interested in the correlations between the variables and their distributions. The descriptive statistics (such as means, etc.) can be also obtained with the linear regression analysis, so no need to conduct the Descriptives for now. Again, the scatterplots can reveal some highly problematic data points prior conducting the analysis itself. What we also strive for is that each predictor (adverts, airplay, attract) would correlate with the dependent variable (sales), but not not with each other (correlated predictors cause the so called multicollinearity problem).

Linear Regression

And now to the multiple linear regression!

Previously, we already computed the regression using only advertisement budget to predict the sales. Now, we will include two more variables: airplay and attract. Because we already know that adverts predict the sales, it is certain than a new model will explain at least the same amount of variance of sales. Because of this, we can test whether including the two new predictors explains the album sales better than including just adverts. After we included the three variables as covariates, we can add the adverts to the null model in the 'Model' panel. Now, we will test the new model against the one we fitted moments before (predicting sales by adverts).

The first line of the model summary shows the 'null model'. See that the statistics correspond to the ones we computed earlier (predicting sales by adverts). The second line is about the new model with three predictors. Based on the R2 Change and F-Test of the change, we can see that the new model does significantly better than the previous one. The adjusted R2 of the model also drops only a little, showing robust model (probably not very high overfitting).
